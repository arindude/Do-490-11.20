Project Report

Automated Regime-Based Portfolio Allocation System

Full write-up outlining the data inputs, modeling approach, analytics pipeline, and outcomes for the portfolio allocation system.

Context & Goal

Create a transparent pipeline that converts macroeconomic signals into allocation recommendations with defendable rationale and risk controls.

Methodology

  • Ingest 13 macro indicators (e.g., breakeven inflation, growth, financial conditions) and normalize via custom z-scores.
  • Score regime probabilities and map them to allocation bands across asset classes with exposure caps.
  • Run analytics: returns, volatility, Sharpe/Sortino, drawdowns, and Value at Risk to compare candidate portfolios.
  • Produce investor memos summarizing the regime call, allocation, and associated risks.
  • Build controlled test cases to expose pricing/logic breakdowns and improve reliability.

Tech & Tools

Python (Pandas, NumPy, SciPy, Statsmodels) Excel Underwriting Risk Analytics Investor Communications

Outcomes

Delivered allocation outputs with clear risk/return trade-offs and memos suitable for stakeholders. QA reduced model failure rates by more than 90% on edge-case tests.
